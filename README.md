# To-Do List Application

A responsive and interactive To-Do List application built with TypeScript, focused on professional design, functionality, and smooth animations. Users can add, edit, and delete tasks with ease.

## Features

1. Add tasks to the list.
2. Edit tasks in place.
3. Delete tasks with a smooth animation effect.
4. Responsive design for seamless use across devices.
5. Animation-driven user interactions, implemented in TypeScript.

## Tech Stack
1. TypeScript: The primary language used to build the app, ensuring type safety and better code quality.
2. HTML: For creating the structure of the web page.
3. CSS: For styling and layout of the app.
